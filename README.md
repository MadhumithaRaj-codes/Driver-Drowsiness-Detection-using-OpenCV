# 🚗 Driver Drowsiness Detection using OpenCV (ADAS)

## 📌 Overview
This project implements a real-time driver drowsiness detection system using computer vision and machine learning techniques. It analyzes facial features such as eye closure and yawning to detect fatigue and trigger alerts.

## 📄 Publication
Published in IEEE ICACCS 2024  

📥 [Read the Paper](docs/Driver_Drowsiness_Detection_IEEE_2024.pdf)  
🔗 DOI: https://doi.org/10.1109/ICACCS60874.2024.10716918  

## 🚀 Features
- Real-time face detection using OpenCV  
- Eye Aspect Ratio (EAR) for blink detection  
- Mouth Aspect Ratio (MAR) for yawning detection  
- Real-time alert system for fatigue detection  

## 🛠️ Tech Stack
- Python  
- OpenCV  
- Dlib  
- NumPy
